Pengaturan Periode & Data

Contoh: isi 3 jika data mencakup 3 tahun (mis. 2022, 2023, 2024)

Komponen Laporan Keuangan (Tetap 5)
1Aset 2Liabilitas 3Ekuitas 4Pendapatan 5Beban
Tipe Target
mKurangi kekurangan dari target
pKurangi kelebihan dari target
bTepat target dari dua sisi

m → komponen yang ingin diminimalkan (Beban, Liabilitas)
p → komponen yang ingin dimaksimalkan (Pendapatan, Ekuitas, Aset)
b → ingin tepat sesuai target dua arah

📋

Input Data Laporan Keuangan

Format tiap baris: [nilai periode 1] [nilai periode 2] … [nilai target] | [tipe]

Contoh 3 periode — target Aset 500 juta: 45 65 90 500 | m

Isi 5 baris urutan: Aset · Liabilitas · Ekuitas · Pendapatan · Beban

Import Excel: Kolom = nilai tiap periode + kolom target + kolom tipe (m/p/b). 5 baris = 5 komponen.
